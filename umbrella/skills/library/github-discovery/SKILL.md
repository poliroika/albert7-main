## GitHub Discovery

Use GitHub as prior art, not as blind code paste.

Search pattern:
- Start with task-domain terms.
- Add stack terms from the requested implementation.
- Prefer maintained, permissively licensed examples.
- Store useful references in palace memory with why they matter.
- Verify relevance before using a reference in the plan.

If search returns weak matches, say so and use them only as loose architectural hints.
