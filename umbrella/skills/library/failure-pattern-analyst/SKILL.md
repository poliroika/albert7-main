## Failure Analysis

Write lessons that improve future runs.

Capture:
- Triggering symptom.
- Root cause in tool, prompt, memory, phase, or workspace code.
- Detection signal that would have caught it earlier.
- Durable rule or test to prevent recurrence.

Separate user task defects from automation defects.
