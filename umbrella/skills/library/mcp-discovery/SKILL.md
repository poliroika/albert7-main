## MCP Discovery

Use MCP discovery after understanding the task domain.

Steps:
- Query for domain-specific tools.
- Check whether the result directly reduces implementation risk.
- Record useful candidates with install requirements and constraints.
- Skip installation unless the tool is clearly necessary.

An empty discovery result is acceptable evidence when logged.
