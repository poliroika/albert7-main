# News Cards AI — шаблон задания

Воркспейс намеренно содержит только исходники постановки. Реализацию добавляет исполнитель по этим файлам.

## Исходные материалы

| Файл | Назначение |
|------|------------|
| `пайплайн_автоматические_карточки.docx` | ТЗ, этапы пайплайна, поля карточек, ограничения, тон, структура слайдов. |
| `Template_новости недели.pptx` | Эталон оформления и структуры итоговой презентации. |

## Цель (кратко)

Сервис или пайплайн, который по пользовательским тезисам новостей формирует презентацию в духе шаблона и требований из docx. Детали сценариев, полей, промптов и критериев приёмки — **только из docx** (и визуально из pptx).
По итоу должен быть сервис который по тезисам пользователей в web ui генерирует полноценные карточки из шаблонов и требований в docx, также проверяет переди этим тезисы через агентов что дописать необходимо какие ссылки всатвить и так далее.
